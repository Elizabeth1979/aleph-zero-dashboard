Quick Start
aleph
Opens the interactive CLI. The aleph alias is configured in your shell.
Interfaces
Interactive CLI
Type aleph in terminal
Discord
Message Aleph Zero bot
HTTP API
POST localhost:3141/chat
One-shot CLI
npm run cli -- <cmd>
Agent
ask default
Ask the agent anything. This is the default — just type your question without a command.
aleph > what do my notes say about color contrast? # Uses search_vault, search_memory, or answers directly
health status
Check agent, Ollama, and server status.
aleph > health Agent: running (pid 12345) Ollama: online (3 models) Model: gemma4:e4b Server: http://localhost:3141
Tasks
tasks t, todo
List all tasks with urgency and tags.
add
Add a new task.
add <description> [--urgent] [--tags work,private]
aleph > add Review PR for button component --tags Work
Memory
memory mem, m
Show MemPalace status — total drawers, wings, and rooms.
Utility
help h, ?
Show all available commands.
clear cls
Clear the screen and reprint the header.
exit quit, q
Exit the interactive CLI.
